Command line interface
=========================

.. _usage:

Usage
======

.. code-block:: bash

    $ elevator-cli
    usage: elevator-cli [-h] [-c CONFIG] [-t PROTOCOL] [-b ENDPOINT]

    Elevator command line client

    optional arguments:
      -h, --help            show this help message and exit
      -t PROTOCOL, --protocol PROTOCOL
      -b ENDPOINT, --endpoint ENDPOINT
