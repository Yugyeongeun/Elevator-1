Elevator

A Minimalistic database engine based on levelDB as a backend, and thrift as RPC message protocol.

Overview

Extensively 

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Dependencies

Elevator relies on three pillars : ZeroMQ, levelDB and Thrift. The project fabfile has a rule to automatically build it on your system. Simpy use : 

$ fab install_dependencies

Installation

$ pip install fabric
$ fab install_requirements
$ python setup.py install
